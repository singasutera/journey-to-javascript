Pop Quiz!

1. Briefly explain what JavaScript is.

JavaScript is a programming language that is used to make web pages interactive.


2. What methods/property are there to display a message in JavaScript?

innerHTML: it's a property used to access or modify the HTML content within an element. Use the method getElementById().

alert(): displays a pop-up dialog box with a message to the user.

document.write(): used to write a string of text to a document stream opened by document.open().

console.log(): displays data in the console.


3. What is a variable? How do you declare and initialize it?

A variable is a container for storing data values. You can declare and initialize it by using the var keyword.

Example: var number = 9;


4. What are the data types in JavaScript? Explain each one.

Number: any number, including numbers with decimals: 4, 8, 15, 16, 23, 42, 108.

  There are two types: integer for whole numbers, and float for decimal numbers.

String: any text inside double or single quotes: "Hello World!"

Boolean: true or false

Undefined: a variable that has been declared but not assigned a value

Null: a variable with no value


5. What is a conditional statement? Provide an example.

A conditional statement is a set of rules performed if a certain condition is met.

Example:

if (x > 9) {

 document.write("x is greater than 9");

}


6. What is the output of the code below?

var name = "Indigo Sonora"

var number = 0;

for(var i=0; i<name.length; i++) {

 num++

}

document.write("The number is: " + number);

The number is: 13


7. Below is declared and initialized an array:

var arr = array(11, 22, 33)

Write a command to add 9 as the last value of the array!

Answer:

8. What is a function? Write a function with a parameter and return statement, then call it!

A function is a block of code designed to perform a particular task.

To call a function, we simply type the name of the function, including the parantheses, and parameters when specified.

Example:

function addition(x,y) {

 var result = x + y;

 return result;

}
